# Projeto-02-CSS

# 🎬 Clone de Interface do YouTube (HTML + CSS) - Confira no link: https://eduardofernandes01.github.io/Projeto-02-CSS/

## 📌 Sobre o Projeto

Este projeto consiste na reprodução da interface de exibição de vídeos do YouTube, desenvolvido com foco em **HTML** e **CSS**, com ênfase na utilização da propriedade **Flexbox** para organização e distribuição dos elementos em tela.

O objetivo principal foi praticar conceitos de layout moderno, estruturação semântica e estilização responsiva, simulando um ambiente real de aplicação front-end.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts (Roboto)

---

## 🎯 Principais Funcionalidades

* Estrutura de página semelhante ao YouTube
* Player de vídeo incorporado
* Seção de informações do canal
* Botões de interação (like, compartilhar, download)
* Lista lateral de vídeos sugeridos
* Layout organizado utilizando Flexbox

---

## 📐 Uso do Flexbox no Projeto

O **Flexbox** foi a principal ferramenta utilizada para construção do layout da aplicação, permitindo organizar os elementos de forma eficiente e responsiva.

### 🔧 Onde foi aplicado:

* Alinhamento do header (logo, busca e ações do usuário)
* Organização da seção principal (vídeo + sugestões)
* Distribuição dos elementos do canal (logo, nome, botão e ações)
* Layout das sugestões de vídeos (thumbnail + descrição)
* Estrutura vertical e horizontal de textos

---

## 💡 Vantagens do Flexbox

A escolha pelo uso do Flexbox trouxe diversos benefícios durante o desenvolvimento:

### ✅ 1. Facilidade de Alinhamento

Permite alinhar elementos vertical e horizontalmente com poucas linhas de código, utilizando propriedades como:

* `justify-content`
* `align-items`

---

### ✅ 2. Distribuição Inteligente de Espaço

Com o uso de propriedades como:

* `flex: 1`
* `gap`

foi possível fazer com que elementos ocupassem automaticamente o espaço disponível, evitando o uso excessivo de tamanhos fixos.

---


### ✅ 3. Redução de Código

Comparado a métodos antigos (como float ou position), o Flexbox reduz significativamente a complexidade do CSS.

---

### ✅ . Organização em Eixos (Row e Column)

A possibilidade de trabalhar com:

* `flex-direction: row`
* `flex-direction: column`

permitiu estruturar conteúdos tanto na horizontal quanto na vertical de forma simples e intuitiva.
